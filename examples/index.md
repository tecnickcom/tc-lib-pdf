# tc-lib-pdf Examples

- [000_example_overview.php](000_example_overview.php) : Generic overview demonstrating a wide range of tc-lib-pdf features.
- [001_example_invoice.php](001_example_invoice.php) : Invoice example with custom XMP metadata (Factur-X / ZUGFeRD).
- [002_example_font_dump.php](002_example_font_dump.php) : Font dump example.
- [003_example_persian_arabic.php](003_example_persian_arabic.php) : Persian and Arabic text example.
- [004_example_page_formats.php](004_example_page_formats.php) : Page formats example.
- [005_example_header_footer.php](005_example_header_footer.php) : Custom page header and footer that automatically repeats on every new page.
- [006_example_minimal.php](006_example_minimal.php) : Minimal example.
- [007_example_signature_basic.php](007_example_signature_basic.php) : Basic detached CMS (PKCS#7) signature with visible signature fields.
- [008_example_signature_timestamp.php](008_example_signature_timestamp.php) : Detached CMS signature with RFC 3161 TSA timestamp configuration.
- [009_example_signature_ltv.php](009_example_signature_ltv.php) : Detached CMS signature with LTV validation material embedding (/DSS and /VRI).
